# 🎮 Simon Says Game

A simple **Simon Says Game** built using **HTML, CSS, and JavaScript**.  
The game tests your memory by showing a sequence of colors that you need to repeat correctly. Each level increases the sequence length.

---

## 🚀 Live Demo
[Click here to play](https://simon-says-game-7k20.onrender.com/)

---

## 🎯 Features
- Start game with keypress or mouse click  
- Random sequence generation  
- Flash animations for buttons  
- Score tracking with Game Over screen  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript

---

## ▶️ How to Play
1. Press any key or click outside the buttons to start.  
2. Watch the sequence of flashes.  
3. Repeat the sequence by clicking buttons in the same order.  
4. Game ends when you press the wrong button.  
5. Your **score = highest level reached**.
